# My Open AAC

My Open AAC is a privacy-first, open-source Progressive Web App (PWA) designed to provide non-verbal individuals with a highly responsive, customizable, and completely offline augmentative communication canvas. 

By eliminating cloud dependencies, the application ensures absolute operational reliability and data privacy. Every component—from the multi-threaded WebAssembly text-to-speech engine to the hardware audio processing graphs, local image compressor pipelines, and microphone voice clip recorders—runs entirely client-side directly on the host device.

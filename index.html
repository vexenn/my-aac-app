❓ AAC Application Guide

🔐 Unlocking Admin Mode

To prevent accidental edits, press and hold the Settings (Hold 5s) button in the bottom right corner for exactly 5 seconds to unlock this control drawer.

➕ Adding & Editing Words

While in Admin Mode, click any empty placeholder cell (e.g., R3 C4) to create a word. Type a label, upload a picture/symbol, or record your own voice clip using the microphone! Click an existing word to edit its contents or move it by dragging.

🗑️ Deleting & Restoring Tiles

To delete a word, click it in Admin Mode and press the red Delete Selected Tile button at the bottom of the drawer. If you make a mistake, click ↩️ Undo Edit under Layout History to instantly restore deleted items.

📺 Fullscreen Protection Mode

Click Enter Fullscreen Mode inside the Admin drawer before giving the tablet to a child. This hides the operating system navigation bars so they cannot accidentally exit the app.

♿ Assistive Switch Scanning

The app supports Bluetooth buttons and spacebar scanning! Pressing the Spacebar or a connected game controller button automatically scans through available communication tiles.
